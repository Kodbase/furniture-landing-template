# 🪑 Furniture Website Template

Modern, hızlı ve tamamen özelleştirilebilir bir **mobilya / koltuk web sitesi template’i**.  
Küçük ve orta ölçekli işletmeler için hızlı kurulum ve kolay düzenleme odaklıdır.

---

## ⚡ Özellikler

- React + Vite altyapısı
- Modern ve responsive tasarım
- Component tabanlı mimari
- Config & data odaklı yapı (kolay içerik yönetimi)
- Tek sayfa (landing) yapısı
- Performans odaklı sade kod yapısı

---

## 🚀 Kurulum

```bash
git clone <repo-link>
cd project
npm install
npm run dev
